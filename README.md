# Job Automation

Automated job discovery system built with Python and Playwright.

## Features

* Scrapes jobs from Naukri.com
* Filters by keywords, location, and experience
* Exports results to CSV
* AI-powered job scoring using Claude Haiku (next phase)

## Tech Stack

* Python
* Playwright
* GitHub
* Anthropic Claude API
